
Скрипт : read_pkl.py                                                                                                                                                                        
   
  Использование:                                                                                       
  # Показать всё содержимое                                                                                                                                                                                 
  python scripts/read_pkl.py path/to/file.pkl                                                                                                                                                           
  # Показать только первые 5 элементов
  python scripts/read_pkl.py path/to/file.pkl -n 5

  Скрипт автоматически определяет тип данных (list, dict и т.д.) и выводит содержимое в читаемом формате.
